# 💰 Sistema de Orçamento em Python

Sistema desenvolvido em Python para criação e gerenciamento de orçamentos, com geração de PDF.

## 🚀 Funcionalidades

- Cadastro de empresa
- Cadastro de clientes
- Criação de orçamentos
- Listagem e exclusão de orçamentos
- Geração de PDF com dados do orçamento

## 🛠️ Tecnologias

- Python
- JSON (armazenamento de dados)
- ReportLab (geração de PDF)

## 📂 Estrutura

- `main.py` → Menu principal
- `empresa.py` → Cadastro da empresa
- `cliente.py` → Cadastro de clientes
- `orcamento.py` → Lógica de orçamentos
- `banco.py` → Persistência em JSON

## ▶️ Como executar

```bash
python main.py
